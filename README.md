nlp-ci
======
Continue Integration (CI) for Natural Language Processing.

## Requirements
This script is tested on Ubuntu 14.04. It is recommended to use these scripts.

Minimum requirements:
- Python 2.7.6+
- pip 1.5.4+
- Fabric 1.8.2+

## Installation
1. Prepare linux server.
2. Run install.sh
